You right-click on a folder and there's no security tab!

Then you probably have 'Use simple file sharing' enabled.

Go to Tools > Folder Options > View, scroll to the bottom of the list and uncheck 'Use simple file sharing'

The security tab should now appear in the directory's properties dialog box.
